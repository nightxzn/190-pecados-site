Política de Privacidade

Bot 190 PECADOS • Atualizado em 2026

1. Esta política descreve o uso de dados pelo bot.

2. Apenas dados necessários ao funcionamento são coletados.

3. IDs de usuários, servidores e canais podem ser armazenados.

4. Mensagens privadas não são lidas nem armazenadas.

5. Nenhuma informação sensível é coletada.

6. Dados são usados apenas para fins operacionais.

7. Dados não são compartilhados com terceiros.

8. Medidas de segurança são aplicadas.

9. Nenhuma segurança absoluta é garantida.

10. Dados podem ser removidos mediante solicitação.

11. Logs técnicos podem existir.

12. Logs não são usados para vigilância.

13. O bot segue diretrizes do Discord.

14. Nenhum dado é vendido.

15. O uso implica consentimento.

16. Alterações podem ocorrer.

17. Alterações entram em vigor imediatamente.

18. O bot pode ser desativado.

19. O uso é opcional.

20. Dúvidas devem ser tratadas oficialmente.
